## Technological Landscape

The gold-standard for motion analysis are proprietary marker-based systems as Vicon, Qualisys, and OptiTrack, or commercial markerless systems such as Theia3D. These tools are costly, closed-source, and difficult to modify, audit, or extend. OpenCap provides an  open-source alternative, but its workflow is tied to specific hardware, cloud processing, and a fixed analysis pipeline. Pose2Sim supports functions as a processing workflow rather than an end-to-end capture and analysis system.

## AI Applications

FreeMoCap is fundamentally a tool to support astandalone desktop application for high throughput multi-camera AI/ML image analysis. See the FreeMoCap Foundations Track 2 proposal for details on how we plan to expand users ability to build custom AI/ML pipelines.
